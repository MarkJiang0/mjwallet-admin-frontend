## Quick Start

### Clone project

```bash
git clone https://github.com/MarkJiang0/mjwallet-admin-frontend.git
```

### Install dependencies

```bash
npm install
```

### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```
